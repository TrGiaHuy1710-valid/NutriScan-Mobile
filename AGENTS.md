# AGENTS.md - HealTrack Daily

This repository uses `docs/AGENT.md` as the detailed agent harness.

Current product direction:
- Main bottom navigation is Home, Foods, Workout, Profile.
- The old visible Meals tab is removed.
- Add Meal and Meal Log are hidden/internal nutrition flows opened from Dashboard, scan confirmation, and food recommendations.
- Foods / Discover is a visible main screen focused on healthy food discovery, mock recommendations, scanned products, and bought-today items.
- MVP remains UI-first with mock/local data only.

Always read `docs/AGENT.md` and `docs/healtrack_plan_harness/*.md` before coding.
